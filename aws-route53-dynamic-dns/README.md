# Custom Home Assistant Add-on: AWS Route 53 Dynamic DNS

Automatically update your AWS Route 53 IP address.

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield] ![Supports armhf Architecture][armhf-shield] ![Supports armv7 Architecture][armv7-shield] ![Supports i386 Architecture][i386-shield]

## About

[AWS Route 53][aws] is one of many services offered by Amazon Web Services. This add on can automatically update your Route 53 DNS record if your Internet Service Provider assigns you a dynamic (changing) public IP Address.
